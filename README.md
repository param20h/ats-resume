# AI Resume Builder with ATS Scoring

An AI-powered resume builder that helps you create professional resumes and check their ATS (Applicant Tracking System) compatibility score.

## Features

- AI-powered resume generation
- ATS score analysis against job descriptions
- Identify matching and missing keywords
- Get personalized suggestions to improve your resume
- Download your resume as PDF

## Tech Stack

- **Frontend**: React.js with styled-components
- **Backend**: Node.js with Express
- **AI Integration**: Google's Gemini AI API

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Google Gemini API key

## Installation

1. Clone the repository:
```
git clone <repository-url>
cd ats-resume-builder
```

2. Install dependencies for both client and server:
```
npm run install-all
```

3. Create a `.env` file in the server directory:
```
cd server
touch .env
```

4. Add your Gemini API key to the `.env` file:
```
PORT=5000
GEMINI_API_KEY=your_gemini_api_key_here
```

## Running the Application

1. Start both client and server:
```
npm run dev
```

This will start:
- Client at http://localhost:3000
- Server at http://localhost:5000

## Usage

1. **Create a Resume**:
   - Fill in your personal information, work experience, education, and skills.
   - Click "Generate Resume" to create your professional resume.

2. **Check ATS Score**:
   - Paste a job description
   - Paste your resume text or use the one generated by the AI
   - Get detailed analysis with matching keywords, missing keywords, and suggestions

3. **Download Resume**:
   - Preview your generated resume
   - Download it as a PDF

## Project Structure

```
ats-resume-builder/

```

## Getting a Gemini API Key

1. Go to the [Google AI Studio](https://ai.google.dev/)
2. Create a new API key
3. Add the key to your `.env` file
